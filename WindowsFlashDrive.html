Can you install a version of Windows in a flash drive?


Difficulty: Moderate


Step 1

Find a Computer with Windows Vista or later. You will need to install a disk image from Microsoft.
Don't worry, I also attached some disk images to make life more easier.


Step 2

Install WinToUSB, Another thing I have attached to the webpage. It is freeware and we will be using it
to flash the Windows image onto the Flash Drive. Note You can only add Windows 7, Windows 8, Windows 8.1, and Windows 10 (From 1507 to 1909) to a flash drive


Step 3

Go through WinToUSB and make sure you have the right drive connected.


Step 4

After WinToUSB is finished, you need to reboot and get to the BIOS of your computer (Tip computers
use different F keys to get to the BIOS, Try F12, F10, F2, and Escape)


Step 5

Change the boot option making the "Removable Media" first in line. I recommend having the "Hard Drive" option second in line.


Step 6

Save and Reboot.


Step 6

Do setup just like if you just got a new PC. After you complete the setup, you will be at the Desktop!
