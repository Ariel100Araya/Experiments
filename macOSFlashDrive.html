Can you install a version of macOS in a flash drive?


Difficulty: Easy


Step 1

Install the installation media for macOS. You can do this by going to the App Store and
Downloading the install media or by going to Apple's website and finding install macOS [version name]


Step 2

Open the install media and go through the terms. When you get to the "select disk for installation" screen, stop


Step 3

Connect your flash drive and use Disk Utillity to format or partion it. Note You
have to format the flash drive as HFS+ (or MacOS Jounaled Extended) and make sure to click GUID partition map.


Step 4

Back on the install media software, click on the freshly made disk and click continue.


Step 5

When the system restarts, press the option key. After you do, use the arrow keys to select the disk.


Step 6

After you wait for the Operating System to restart, you can start setting up your computer as normal!
